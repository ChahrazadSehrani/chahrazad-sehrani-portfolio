# Project — Generative AI for Helicopter Image Generation

## Context

Industrial project with Airbus Helicopters focused on generating realistic and controlled helicopter visuals using generative AI.

## Objective

The project involved developing and fine-tuning generative deep learning models to generate high-quality helicopter images.

## Techniques

- Stable Diffusion
- FLUX
- GANs
- Prompt engineering
- Fine-tuning
- Controlled image generation
- Image generation model optimization

## Keywords

`Stable Diffusion` · `FLUX` · `GAN` · `Generative AI` · `Prompt Engineering` · `Fine-tuning`
