# Project — Computer Vision for Human–Robot Collaboration

## Context

During my internship at CRAN, I developed an AI pipeline for detecting, segmenting, and tracking people, robots, and objects using an Intel depth camera.

## Objective

The objective was to support perception capabilities in human–robot collaboration scenarios.

## Pipeline

- Data collection
- Data preprocessing
- Object detection
- Segmentation
- Tracking
- Depth-camera-based perception
- Model experimentation

## Keywords

`Computer Vision` · `YOLO` · `Deep Learning` · `Segmentation` · `Tracking` · `Human–Robot Collaboration`
